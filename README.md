# BattleArenaGame
A console-based battle game in Java using OOP concepts like inheritance, abstraction and encapsulation.
# Features
* Multiple character classes:

  * Warrior
  * Mage
  * Archer
* Randomized attack damage system
* Turn-based battle mechanics
* Health management system
* Winner detection
* Console-based interactive gameplay

## Technologies Used

* Java
* OOP Principles
* Scanner Class
* Math.random()

## Gameplay

* Players create characters
* Characters attack each other turn by turn
* Random damage is applied during attacks
* The battle continues until one player's health reaches 0
